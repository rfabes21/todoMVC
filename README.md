# Marionette HTML Application Foundation

## Features

* Compass
* Saas
* RequireJS
* Marionette
* Backbone
* Handlebars
* Underscore

## Requirements

* [Node.js](http://nodejs.org/download/)
* RequireJS
* Compass

Download and install Node.js then run the following.

```
gem install compass sass
sudo npm install -g requirejs
```


## Quick start

```
make all
```
